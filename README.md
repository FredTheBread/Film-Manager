# Film Manager 

This is a desktop application built with film students in mind to help them plan and organize upcoming film shoots.  
It’s written in **Java** and built with **JavaFX** for a simple and easy-to-use UI.


---

## Overview

Film Manager is designed to make pre-production planning easier for film students and small crews.  
Instead of having to use multiple spreadsheets and notes, they can manage their shoots in one place with a dedicated desktop app.
It simplifies the process of planing and shooting a movie by having all the needed parts for crew members, actors, their roles, film dates and the script of the movie.
In addition to this the simplicity of saving all the given inputs to a txt file and sharing it to other crew members or actors.

Inteded use cases:

- Planning upcoming film shoots for class projects
- Keeping track of shoot dates, locations, and notes
- Having a simple, offline tool for pre-production planning

---

## Features

- **Shoot planning:** Create and manage entries for upcoming film shoots.
- **Organized information:** Store important information such as title, date, location, and notes for each shoot.
- **Desktop UI with JavaFX:** An easy to understand graphical interface.
- **Local application:** Runs fully on your machine — ideal offline environments and for sharing locally.

---

## Tech Used

- **Language:** Java
- **UI Framework:** JavaFX
- **Build/Project Setup:** NetBeans/Ant-style project layout (`build.xml`, `nbproject`, `dist/`)

---

## Usage

1. **Create a new film shoot**  
   Enter details like shoot title, date, time, location, and other important notes.

2. **View all upcoming shoots**  
   See all saved film shoots to see what is coming up.

3. **Edit or remove shoots**  
   Update all details as the movie evolves.

This interface is ideal for students managing multiple assignments or small productions.
